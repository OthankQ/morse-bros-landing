Fun Facts About Morse Code

Morse code was invented by Samuel Morse. That's where the 'Morse' part of Morse code comes from.

Mr. Morse was also a painter who painted the portrait of Marquis de Lafayette.

The universal SOS signal was created because it was very easy to remember in Morse code!

You can actually speak Morse code. The dot is pronounced 'di', and the lines are pronounced 'dah'.
